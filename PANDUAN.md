1. Sebelum menjalankan localhost dari project yang saya buat, mohon untuk memastikan database dengan nama db : toko_retail.
2. Untuk membuatnya secara otomatis bisa eksekusi database/init.sql dan databasenya sudah selesai dibuat.
3. ketikkan npm run dev dan masukkan dengan link localhost yang diberikan.
4. Kita akan diarahkan ke halaman dashboard Admin yang isinya Total Produk, Total Stock, Pembelian Selesai dan Pembelian Dibatalkan.
5. Didalam dashboard Admin kita bisa melakukan Input Pembelian Baru, Proses Pembelian, Cancel Pembelian dan cek stock.
6. Mungkin ini yang bisa saya jelaskan alur/panduan singkat mengenai program saya semoga bisa running.
